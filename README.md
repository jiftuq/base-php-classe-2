## Installation de WAMP sous Windows

N'allez pas sur le site grand public:

https://www.wampserver.com/

Allez sur la version `dev` :

https://wampserver.aviatechno.net/?lang=fr

### Etape 1

Vérification des fichiers nécessaires : [Vérifications paquetages VC++ installés](https://wampserver.aviatechno.net/files/tools/check_vcredist.exe).

Si vous ne devez rien installer, passez à l'étape 3

### Etape 2

Installez les fichiers manquants contenus dans ce paquet : [All VC Redistributable Packages (x86_x64) (32 & 64bits)](https://wampserver.aviatechno.net/files/vcpackages/all_vc_redist_x86_x64.zip)

Quand tout est installé, passé à l'étape 3

### Etape 3

Installez Wamp [Wampserver 3.4.0 64 bit - Apache 2.4.65 - PHP 8.0.30/8.1.33/8.2.29/8.3.28/8.4.15/8.5.0 - MySQL 8.4.7 - MariaDB 11.4.9](https://wampserver.aviatechno.net/files/install/wampserver3.4.0_x64.exe)
